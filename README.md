# Yandex images downloader

A very simple example that downloads visible images and save list links images


### Installing

 Python: 
```
pip install selenium
```
Windows:
```
downloads .exe files
```

### Compatibilty
* Python 3.4+
* Selenium 3.4.0+
* Chrome 80.1+
